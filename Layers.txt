6 TYPES OF LAYERS CREATED
================================================================================

1. VECTOR LAYERS
   Raw geographic features from OpenStreetMap (points, lines, polygons)
   
2. RASTER LAYERS
   Satellite data & terrain (NDVI, NDWI, NDBI, LST, DEM, Slope, Aspect)
   
3. PROXIMITY RASTERS
   Distance analysis from features (Distance from roads, airport, rail, etc.)
   
4. RECLASSIFIED LAYERS
   Standardized 1-5 suitability scores for all analysis layers
   
5. MASK LAYERS
   Binary constraints (0/1) for hard restrictions on development

6. BUFFER LAYERS
   Vector buffers around features for constraint analysis

7. FEATURE MAPS
   Final composite output maps showing IT park suitability results
